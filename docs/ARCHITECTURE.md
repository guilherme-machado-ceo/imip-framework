## IPII Flow

```
Input token
    |
O: Operationalize -> GuruMatrix 5D coordinate (i,j,k,t,l)
    |
P: Process -> compute significance profile f(A) in [0,1]^6
    |
D: Distribute -> broadcast profile (NOT raw vocabulary) to instances
    |
I: Infer -> LLM generates candidates, rho1-rho6 evaluates
    |
N: Incide -> select best, update GuruMatrix, project output
    |
Iterate until PI(A) = [f(A)]^(1/pi) > threshold
```
